# 📋 Gerenciador de Tarefas

Projeto desenvolvido como atividade acadêmica com o objetivo de praticar conceitos de desenvolvimento web utilizando HTML, CSS e JavaScript.

## 🚀 Funcionalidades

- ✅ Adicionar tarefas
- ✏️ Editar tarefas
- ❌ Excluir tarefas
- ✔️ Marcar como concluída
- 🔍 Buscar tarefas por título
- 🎯 Organização por prioridade (Baixa, Média e Alta)
- 💾 Armazenamento local com LocalStorage
- 📊 Contador de tarefas

## 🖥️ Demonstração

🔗 Acesse o projeto online:  
https://izabelaxavier.github.io/lista-de-tarefas/

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foram praticados:

- Manipulação de DOM
- Eventos em JavaScript
- Uso de LocalStorage
- Organização de código
- Estruturação de interface (UI)
